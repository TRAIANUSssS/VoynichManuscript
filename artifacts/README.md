# Artifacts

This directory will store generated experiment outputs such as CSV, JSON, PNG, HTML, reports, and logs.

Artifacts should be grouped by experiment, for example `artifacts/exp001/`.

Current artifact sets:

- `exp001/` - baseline statistics outputs for `exp-001_baseline-statistics_active_2026-06-21`.
- `exp002/` - word-position pattern outputs for `exp-002_word-position-patterns_active_2026-06-21`.
- `exp002b/` - cleaned IVTFF parser rerun outputs and old-vs-cleaned comparisons for `exp-002b_clean-ivtff-parser-rerun_active_2026-06-21`.
