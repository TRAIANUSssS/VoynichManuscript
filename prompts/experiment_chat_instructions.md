# Experiment Chat Instructions

## Purpose

Experiment chats should focus on one experiment or one tightly scoped analysis question.

## How to Start an Experiment Chat

Start by pasting the relevant experiment file, current protocol if any, dataset notes, and method constraints.

## What Context to Provide

Include:

- experiment ID and status;
- related hypotheses;
- data source and version;
- current protocol;
- expected outputs;
- known limitations.

## How to Prepare a Codex Task

Use `prompts/codex_task_packet_template.md`. Be explicit about files to read, files to update, and what must not change.

## How to Return Results to the Main Chat

Return a concise session summary with links to updated files, artifacts, limitations, and open questions.

## Required Final Session Summary

Every substantial experiment chat should end with a summary suitable for `logs/session_summaries.md`.
