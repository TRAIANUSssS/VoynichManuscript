# Statistical Tests

## Purpose

Statistical tests should distinguish reproducible evidence from visual pattern matching and anecdotal examples.

## Baselines

Every strong claim should be compared against simple baselines where possible.

## Comparing Sections

Section comparisons require documented section metadata and controls for sample size.

## Comparing Transcriptions

If multiple transcriptions are used, differences must be documented and tested separately.

## Comparing Generated Text with Voynichese

Generated-text comparisons must save generator code, parameters, random seeds if applicable, and outputs.

## Multiple Comparisons Warning

Exploring many metrics increases false-positive risk. Record exploratory work separately from planned tests.
