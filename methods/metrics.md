# Metrics

## Token Count

Total number of token instances after the selected tokenization policy.

## Type Count

Number of unique token forms.

## Type-Token Ratio

Type count divided by token count. Interpret cautiously because it is sensitive to corpus size.

## Word Length Distribution

Distribution of token lengths under the selected transcription and normalization policy.

## Glyph Frequency

Frequency of individual transcription characters or glyph units.

## Token Frequency

Frequency of token forms.

## Entropy

Information-theoretic summary of uncertainty in a distribution. Implementation details must be recorded.

## Positional Statistics

Counts and proportions by token, line, folio, section, or other documented position.

## Section-Level Comparison

Comparison of metrics across manuscript sections, only after section metadata is documented.

## Notes

All metrics must record input data, preprocessing, parameters, and limitations.
